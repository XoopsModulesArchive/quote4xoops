Citation pour xoops 2
---------------------
Version 1.2
Merci à WismeriL pour sa contribution (templates, logo, ...)

Installation:
- Copier le contenu du dossier quote4xoops dans le répertoire modules
- Activer le module quote4xoops dans le menu d'administration

Mise à jour: (IMPORTANT)
Il semble qu'il y est un probleme lors de la mise à jours de l'ancienne version de ce module
(C'est peut-être du à un probleme xoops, car la nouvelle version inclus la gestion des templates
et toutes les constantes de messages ont été remaniées pour les faire coller avec les préconisation
de l'équipe de développement)
Pour ne par avoir de probleme durant cette mise à jour, il est conseillé de :
- Faire une sauvegarde de la table quote4xoops (avec phpmyadmin ou autre)
- Désactiver puis désinstaller le module
- Supprimer le dossier quote4xoops (version précédente) du répertoire module
- Copier le contenu du dossier quote4xoops (nouvelle version) dans le répertoire module
- Activer le module dans le menu d'administration
- Restaurer la sauvegarde de la table quote4xoops


Bonus:
Dans le dossier quote4xoops/sql, le fichier citation_fr.sql contient 1413 citations.
Exécuter le via votre outils favoris (eskuel, phpmyadmin,...) après avoir activé le module.
(Avant d'éxécuter ce fichier, bien penser à changer le prefix de la table quote4xoops afin
de le remplacer par celui utilisé par votre site)

Quote for xoops 2 
-----------------
Version 1.2

Installation:
- Copy the quote4xoops folder into your xoops modules folder 
- Activate quote4xoops in your Xoops Module administration.

Update: (IMPORTANT)
- Backup the table quote4xoops (with phpmyadmin or other)
- Inactive and désinstall the module
- Delete folder quote4xoops (old version) from modules folder
- Copy the quote4xoops folder (new version) into your xoops modules folder
- Activate quote4xoops in your Xoops Module administration.
- Restore the content of the table quote4xoops
